Файлы с расширением .html открываются в браузере и состоят из тегов

  • Форматирование текста1
  • Выделение текста
  • Графические элементы
  • Cтруктура страницы

Справочник тегов

I. Теги для форматирвоания текста

  • 1. p - абзац (paragraphe)
  • 2. h1...h6 - заголовки (heading)
  • 3. ul/ol - маркированный и нумерованный списки (unordered/ordered list)
  • 4. li - пункты списка (list item)
  • 5. a - якорь (anchor) подробнее

II. Теги для выделения текста

  • 1. b - жирный (bold)
  • 2. i - курсивный (italic)
  • 3. del - перечеркнутый (delete)

III. Теги для графических элементов

  • 1. hr - линия (horizontal rule)
  • 2. img - картинка (image)

IV. Теги задающие структуру

  • 1. !DOCTYPE html - инструкция для браузера
  • 2. html - заключает в себе все содержимое веб страницы
  • 3. head - для хранения служебной информации о странице
  • 4. body - содержимое документа html
  • 5. section - задает раздел документа
  • 6. div - универсальный контейнер
  • 7. header - шапка сайта
  • 8. nav - меню сайта
  • 9. main - основная информация
